# Instructions to initialize and push local files to a new remote GitHub repository

1. Initialize a local git repository if not already done:
   git init

2. Add the remote origin:
   git remote add origin https://github.com/uzicodes/CV-latex.git

3. Add all files:
   git add .

4. Commit the files:
   git commit -m "Initial commit: add cv.tex and my_CV.pdf"

5. Push to GitHub:
   git push -u origin main

If 'main' branch does not exist, use:
   git push -u origin HEAD:main

---

You can copy-paste these commands in your terminal to upload your files to GitHub.
